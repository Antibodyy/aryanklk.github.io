Graduate-level course focusing on the integration of microprocessors and embedded systems in mechanical engineering applications.

## Topics Covered

* Embedded systems design and programming
* Mechatronics integration
* Sensor and actuator interfacing
* Real-time control systems
* Microcontroller programming

## Course Project

Hands-on project involving the design and implementation of a microprocessor-controlled mechanical system.

## Skills Developed

* Embedded C/C++ programming
* Hardware-software integration
* System-level design
* Control system implementation
