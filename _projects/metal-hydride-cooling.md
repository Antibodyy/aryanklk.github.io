Devised a novel cooling architecture leveraging metal hydride properties to solve high-density heat transfer challenges in fuel cell electric vehicles.

## Project Overview

Developed an innovative cooling system for hydrogen fuel cell electric vehicles (FCEV) using metal hydride (MH) thermal properties. The project focused on reducing auxiliary power consumption while maintaining effective thermal management.

## Key Contributions

* **Novel Cooling Architecture**: Designed an open metal hydride cooling system integrated with the US DoE's FCEV reference model
* **Material Analysis Automation**: Developed custom MATLAB scripts to automate material property analysis
* **Thermodynamic Modeling**: Modeled isothermal pressure-composition behaviors in SIMULINK for various metal hydride materials
* **Performance Achievement**: Reduced AC load by 18.8% compared to conventional cooling systems

## Technologies Used

* Analysis: MATLAB, SIMULINK
* Modeling: Thermodynamic system modeling, heat transfer analysis
* Integration: US DoE FCEV model
* Materials: Metal hydride thermal properties characterization

## Technical Highlights

* Pressure-composition-temperature (PCT) curve analysis
* Thermal management system integration
* Energy efficiency optimization
* High-density heat transfer solutions

## Impact

The 18.8% reduction in AC load directly translates to improved vehicle range and energy efficiency for hydrogen fuel cell vehicles.
